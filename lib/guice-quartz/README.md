# Guice + Quartz Sample App

+ Google Guice 4.0
+ Quartz Scheduler 2.2
+ Servlet 3.0
+ XML File Based Schedule Configuration

## Usage

> git clone https://github.com/halflite/guice-quartz-sample.git
>
> cd guice-quartz-sample
>
> mvn package
